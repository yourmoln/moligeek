## 运行环境安装

### Windows (推荐)

1. 可前往Python官网下载: [官网](https://www.python.org/)
2. 如果Windows版本高于16299(Windows 10 1709)，可尝试使用Winget命令安装Python
```
winget install Python.Python.3.10
```

---

### Linux

---

**Debian & Ubuntu**

```
apt install python
```

**CentOS**

```
yum install python
```

## 使用需求

1. 安装脚本

**安装步骤**

```
git clone https://github.com/yourmoln/moligeek.git
```

2. 运行脚本

**如何运行**

```
python main.py
```

**请在项目根目录下运行该指令**
