### 使用需求

1. python3
```
pkg install python
```
或等待未来推出的Windows版本

2.安装脚本
###### **安装步骤**

```
git clone https://github.com/yourmoln/moligeek.git
```

3.运行脚本
###### **如何运行**

```
python main.py
```
请在项目根目录下运行该指令
